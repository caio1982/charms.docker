charms.docker - a convience library
=========================================

charms.docker is a Python library for interacting with and configuring
a docker daemon. 

With charms.docker you can:

- Configure any docker daemon
- Map and store DOCKEROPTS, and re-render to the appropriate file on the host
- Execute one off docker commands
- Login to a docker distribution/registry
- Stand up complex stacks of application containers with docker-compose
